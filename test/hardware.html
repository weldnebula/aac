Hardware (things you can touch)

Computer hardware is the set of devices that combine to do the computing. On this site, we use the term "computing" to refer to all of the tasks that are essential to producing usable results.

  • Manipulating data and hardware as directed by instructions
  • Keeping data and instructions available for quick access
  • Storing the data before and after it has been manipulated
  • Controlling the computer's essential and non-essential hardware
  • Receiving data and instructions from a user
  • Showing the results of data manipulation to the user
  • Exchanging data with other computers

Central Processing Unit (CPU)

The CPU is the brain of your computer. It comes in an electronic chip — a combination of billions of transistors and other components, clustered together as tightly as possible to make it run as fast as possible. The typical CPU in a desktop computer looks like a 1-2 inch square of dark gray material, its underside bristling with electrical connecting pins. It manipulates data according to instructions submitted to it by other hardware components, then returns the results.

CPUs recognize a very small set of very simple instructions called machine code. Each instruction accomplishes very little, but they execute billions of instructions per second, which produce most of the activity you see on your computer (and a lot of activity you don't see).

A CPU has a basic problem: It contains no instructions for running the computer when you turn it on. Instead, it looks for instructions on a special memory chip that holds the computer’s basic startup and operating instructions.

Read-Only Memory (ROM)

The ROM chip holds the computer’s basic startup and operating instructions.
