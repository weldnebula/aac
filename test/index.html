All About Computers

Speak introduction aloud

Welcome to All About Computers! This is a resource for regular people to learn how computer technology works. All About Computers doesn't use cookies, and has no advertising.

Modern computer technology is everywhere, and computers have become equal to cars in their impact on our lives. In fact, computing devices like mobile phones might now be more essential than cars in helping us to navigate our world.

Just as car owners need to know the rules of the road, and how to interpret the dashboard lights and gauges, check the oil, and change a tire, computer users need to know how to start and stop their devices, use email, protect their systems against data loss and viruses, learn where to find and how to open files, and engage in other basic computing activities.

Computers aren't hard to learn, you don't need a license to operate one, and mistakes (on a personal computer) aren't life-threatening. The challenge of learning how to use a computer is minor, and the benefits are potentially life-changing.

Industry Influences

How we got here

Computer technology has developed over the years as a product of two primary influences:

  • The nature of logic, mathematics, and the physical world — electricity, magnetism, optics, mechanics, materials, and so on.
  • The choices that key industry players have made in deciding how to take advantage of the behavior of logic, mathematics, and the physical world.
    One key choice was the decision to use the binary number system as the basis for computer logic. (And no, you don't have to understand binary numbers to use a computer!) 

Those choices could have been very different, and still produced successful devices that we would be using instead of the computers we have now. But as each successive design decision was applied to the technology (usually for good reason, sometimes not), the computer industry was directed more and more toward producing the technology that we have today.

What this means to you

Computers weren't discovered, they were created. The design decisions that led us here were made by many different people with many different goals. If any aspect of this technology doesn't seem obvious, it's not you. It's the technology. So if you see something on this website that doesn't make sense to you, it might need a better explanation.

What is a Computer?

A comparison

Consider a typical tool like a hammer. Hammers are designed with only one purpose: to hit things. (Some hammers can also pry things apart.)

A computer, on the other hand, is the first tool in history that’s designed to do an unlimited number of things with data. (And it can also use data to control actual devices and tools, including hammers.)

Essential computer tasks

On this site, we use the term "computing" to refer to all of the tasks that are essential to producing usable results. These very generic tasks are common to almost every modern computer, including:

  • Manipulating data and hardware as directed by instructions
  • Keeping data and instructions available for quick access
  • Storing data before and after it has been manipulated
  • Controlling the computer's essential and non-essential hardware
  • Receiving data and instructions from a user
  • Presenting the results of data manipulation to the user
  • Exchanging data with other computers

Breaking it down

Every computer has three categories of components:

Hardware  

(Things you can touch)

Here's the minimum set of hardware components necessary to create a usable computer:

  • Core hardware
    • A central processing unit (CPU) to manipulate data.
    • A read-only memory chip (ROM) to hold the computer’s basic startup and operating instructions.
    • High speed memory chips (RAM) to hold the data and instructions that are currently being used by the CPU.
    • Electronic circuitry to enable the components to communicate with each other. This includes ports for connecting external devices (such as USB ports) and power.
  • An input device to tell the computer what data to process and how to process it.
  • An output device to show you the results of processing,
  • A storage device to hold programs and documents.
  • A communication device to connect the computer to the rest of the world.

Hardware page  

Software  

(Data and instructions)

Here's the minimum set of software components necessary to create a usable computer:

  • The startup instructions (best known as the basic input/output system (BIOS)) that tell the computer how to control its minimum required hardware. 
  • The  operating system, a set of instructions that:
    • Takes advantage of the computer's built-in ability to control the basic hardware
    • Adds the ability to control additional types of hardware
    • Manages storage space and memory usage
    • Facilitates the installation, removal, and user interaction with programs
    • Manages incoming and outgoing data and signals
    • Provides an interface for users to interact with the computer
  • Data (content), which you manipulate using programs. Types of data include documents, spreadsheets, photographs, web pages, database contents, and millions of other things. 
  • Programs (sets of instructions that are also called applications or apps) dedicated to specific tasks (such as emailing, browsing the web, drawing, playing music or video, managing databases, planning projects, analyzing data, and everything else you can do with a computer.

Rules and conventions  

(Ways of doing things)

Because a computer's hardware and software components can be produced by different manufacturers, the components must use agreed-upon rules and conventions to communicate with each other and with other systems.

  • A rule is a way of doing things that is required by an authority (like a speed limit). 
  • A convention is a way of doing things that has become popular (like the way people say "How're you doing?" as a greeting).

This category includes standards, protocols, guidelines, languages, algorithms, patterns, regulations, and formats, including the following:

  • Programming languages
  • Communication protocols
  • Data standards
  • File formats
  • Compression algorithms

You'll find various rules and conventions cropping up in any discussion of computer activity.

Rules and conventions page  

These categories aren't always distinct from each other. Hardware often has some software built into its electronics; software frequently has to comply with rules and conventions; and hardware must comply with certain rules and conventions as well.
